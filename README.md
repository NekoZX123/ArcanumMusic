# Arcanum Music

v1.6.5 (Kyrios Internal)

简洁美观的多平台音乐软件

支持 网易云音乐 / QQ音乐 / 酷我音乐 / 酷狗音乐

内容丰富, 官方来源, 安全保证

由 @NekoZX123 用心制作

made with TypeScript + Vue.js

## 问题修复

- 修复了 Linux 环境下路径分隔符导致的文件无法访问问题
- 修复了从歌曲 / 歌单等卡片开始播放时需要再次手动点击播放按钮的问题

## 新功能

- 用户偏好 (窗口大小、音量等) 自动保存
- 窗口外观自定义
- 深色模式及不同颜色主题
- 简约样式歌词

## 内测版本特别提示

当前应用为内测版本, 可能存在部分问题. 当出现问题影响使用时, 可使用 Ctrl + R 刷新应用
