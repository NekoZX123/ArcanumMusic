# Arcanum Music

v1.8.0 (Kyrios Internal)

简洁美观的多平台音乐软件

支持 网易云音乐 / QQ音乐 / 酷我音乐 / 酷狗音乐

内容丰富, 官方来源, 安全保证

由 @NekoZX123 用心制作

made with TypeScript + Vue.js

## 问题修复

- 修复 QQ 音乐 API 加密脚本导致应用无法加载的问题

## 新功能

- 添加歌曲名称点击复制功能
- 更新应用图标
- 更新桌面歌词样式

## 内测版本特别提示

当前应用为内测版本, 可能存在部分问题. 当出现问题影响使用时, 可使用 Ctrl + R 刷新应用
