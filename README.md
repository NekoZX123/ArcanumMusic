# Arcanum Music

v1.7.12 (Kyrios Internal)

简洁美观的多平台音乐软件

支持 网易云音乐 / QQ音乐 / 酷我音乐 / 酷狗音乐

内容丰富, 官方来源, 安全保证

由 @NekoZX123 用心制作

made with TypeScript + Vue.js

## 问题修复

- 修复了歌单内容中歌曲名称显示不全的问题
- 修复了歌词样式不随设置自动变化的问题
- 修复动态歌词背景无法提取主色的问题

## 新功能

- 在歌词界面采用半透明控制按钮
- 更新了应用界面的图标设计

## 内测版本特别提示

当前应用为内测版本, 可能存在部分问题. 当出现问题影响使用时, 可使用 Ctrl + R 刷新应用
