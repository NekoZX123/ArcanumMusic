# Arcanum Music

v1.6.7 (Kyrios Internal)

简洁美观的多平台音乐软件

支持 网易云音乐 / QQ音乐 / 酷我音乐 / 酷狗音乐

内容丰富, 官方来源, 安全保证

由 @NekoZX123 用心制作

made with TypeScript + Vue.js

## 问题修复

- 完善了 Linux 环境下的应用打包配置
- 修复了 Linux 环境下无法显示托盘图标的问题
- 修复了问候语有时显示为 `undefined` 的问题

## 新功能

- 主窗口隐藏至系统托盘功能
- 托盘右键菜单可控制播放器
- 更新了设置文件储存方式

## 内测版本特别提示

当前应用为内测版本, 可能存在部分问题. 当出现问题影响使用时, 可使用 Ctrl + R 刷新应用
